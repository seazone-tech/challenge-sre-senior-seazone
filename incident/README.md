# Evidencias Do Incidente

Este diretorio contem evidencias simuladas de um incidente no servico `reservation-api`.

Use os arquivos como se fossem dados coletados durante uma investigacao inicial. Eles foram desenhados para permitir uma causa raiz provavel sem acesso a AWS, EKS ou sistemas internos.

Arquivos:

- `timeline.md`: linha do tempo resumida.
- `alerts.md`: alertas recebidos durante o incidente.
- `app-logs.jsonl`: logs simulados da aplicacao.
- `kubectl-events.txt`: eventos simulados do Kubernetes.
- `kubectl-describe-pod.txt`: descricao simulada de um pod afetado.
- `metrics-snapshot.csv`: metricas sinteticas.
- `deployment-diff.md`: mudanca recente associada ao incidente.
